Identificación y Comprensión de los Datos
Entrada

Base de datos en excel, archivo CSV o tabla de la base de datos con todas las variables.

Proceso

• Periodicidad de entrega de la base.
• Cantidad de registros.
• Campos que vienen en las bases de datos.
• Variables sociodemográficas.
• Identificación de indicador óptimo de SPH.

Salida

Información del negocio. Base de datos en excel, archivo CSV o tabla de la base de datos con todas las variables. SPH óptimo.

Validación
Entrada

• Modelo entrenado ajustado.
• Base de datos de validación.
• Listado de segmentos definidos.

Proceso

• Prueba del modelo de clasificación con base de datos de validación.
• Uso de Matrices de Confusión, Curva ROC.
• Evaluación de métricas: precisión, sensibilidad, especificidad, kappa, etc.
• Generación de reglas para el modelo de clasificación.
• Validación entre el modelo de clasificación y el modelo de segmentación.

Salida

• Listado de segmentos definidos.
• Reglas del modelo de clasificación.

Generación del perfil
Entrada

• Listado de segmentos definidos.
• Reglas del modelo de clasificación.

Proceso

Identificación de los segmentos y las reglas de clasificación más importantes.

Salida

Perfil de colaboradores con base en los segmentos y las reglas definidas.

Control y Seguimiento
Entrada

Resultados de la implementación del perfilamiento.

Proceso

• Se lleva a cabo control y seguimiento de la implementación de la estrategia y gestión de las base de datos a través de los distintos dash creados para tal fin por parte de área de reporting.
• Se elaboran analíticos internos e indicadores claves para mayor control.
• Se obtienen conclusiones y hallazgos del control y seguimiento para su análisis y retroalimentación.

Salida

Dash, analíticos internos, conclusiones y retroalimentación.
