Depuración de la Base de Datos
Entrada

Base de datos en excel, archivo CSV o tabla de la base de datos con todas las variables.

Proceso

• Eliminar variables con datos insuficientes.
• Imputar datos (media, moda, etc.).

Salida

Base de datos en excel, archivo CSV o tabla de la base de datos con variables para usar en el modelo.

Encuesta
Entrada

• Listado de Asesores con su respectivo e-mail.
• Listado de preguntas en Formulario de Google con encuesta.

Proceso

• Identificación y definición de las preguntas de la encuesta.
• Asignación de opciones para cada pregunta.
• Creación de la encuesta en la herramienta de Google Forms.
• Compartir la encuesta diseñada con los asesores del listado para su respectivo diligenciamiento.
• Seguimiento del diligenciamiento.
• Unificación de resultados obtenidos a través de la encuesta en un archivo único.

Salida

• Archivo de Excel con listado de asesores y preguntas a realizar.
• Creación de la encuesta en la herramienta de Google Forms.
• Encuesta enviada a los destinatarios mencionados.
• Archivo en Excel con consolidado de respuestas.

Transformación de las Variables
Entrada

Base de datos con variables seleccionadas.

Proceso

• Creación de nuevas variables calculadas.
• Agrupamiento de valores de las variables por rangos.
• Transformación de variables categóricas.
• Creación de variables dummies.

Salida

Base de datos con variables transformadas y variables nuevas.

Definir Datos del Conjunto
Entrada

Base de datos definitiva para el análisis.

Proceso

• Selección de datos de entrenamiento.
• Selección de datos de prueba.
• Selección de datos de validación.

Salida

• Base de datos de entrenamiento.
• Base de datos de prueba.
• Base de datos de validación.

Balance de la Muestra
Entrada

• Base de datos de entrenamiento.
• Base de datos de prueba.
• Base de datos de validación.

Proceso

• Undersampling: Tomek y Muestreo.
• Oversampling: Smote.
• Combinación de Undersampling y Oversampling (Smote-Tomek).

Salida

• Base de datos de entrenamiento balanceada.
• Base de datos de prueba balanceada.
• Base de datos de validación.
• Base de datos sin balancear.
